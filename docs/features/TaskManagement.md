# Task Management

Lists running processes, shows resource metrics, and lets operators inspect or terminate tasks.

## Features
- Displays CPU, memory, and I/O usage per process.
- Lets administrators terminate, suspend, or reprioritize problematic jobs.
- Supports filtering or searching to find suspicious binaries quickly.

## Basic Workflow
1. Open the Task Management view.
2. Fetch the current process list from the agent.
3. Act on any process by terminating or adjusting it as required.
