# Registry

Explores and edits Windows registry hives, keys, and values from the controller.

## Features
- Navigates registry trees with search and filtering.
- Reads and writes values securely when permissions allow.
- Creates or deletes keys to adjust persistence or clean up artifacts.

## Basic Workflow
1. Open the Registry workspace for a Windows client.
2. Drill down to the target key and view its values.
3. Apply edits or deletions and verify the change.
