# Startup

Manages autorun entries such as services, scheduled tasks, and run keys for persistence control.

## Features
- Lists autorun entries reported by services, scheduled tasks, and the registry.
- Enables or disables entries to control what runs automatically.
- Filters results to focus on persistence or recent additions.

## Basic Workflow
1. Open the Startup workspace.
2. Review the active autorun entries reported by the agent.
3. Toggle or remove entries to adjust persistence.
