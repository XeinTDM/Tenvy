# File Management

Provides remote filesystem browsing, transfers, and metadata inspection directly from the controller.

## Features
- Navigates directories and shows file metadata without extra tooling.
- Uploads or downloads selected files through the UI.
- Supports deleting, renaming, and permission checks when allowed.

## Basic Workflow
1. Open the File Management panel for a client.
2. Browse to the target directory and inspect contents.
3. Transfer or manipulate files using the provided actions.
