# TCP Management

Shows active TCP listeners and connections so administrators understand the networking state.

## Features
- Lists local listeners with ports, protocols, and owning processes.
- Exposes remote peers for established connections.
- Offers filtering to highlight suspicious endpoints quickly.

## Basic Workflow
1. Launch the TCP Management view.
2. Fetch the listener and connection table from the agent.
3. Inspect any entry and take action if needed.
