# Open URL

Tells the agent to open a browser tab with the supplied link for onboarding or demonstrations.

## Features
- Sends a URL to the agent default browser instantly.
- Logs the request for auditing what was opened.
- Can include focus or timing hints when available.

## Basic Workflow
1. Launch Open URL and provide the target link.
2. The controller sends the command to open the browser tab on the agent.
3. Confirm success in the UI or follow up through chat if needed.
