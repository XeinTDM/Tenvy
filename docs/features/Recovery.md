# Recovery

Creates, uploads, and downloads disk-based archives to preserve data for later analysis.

## Features
- Selects files, directories, or custom paths for archival inclusion.
- Verifies integrity with checksums before uploading.
- Makes archives available for secure downloads once processed.

## Basic Workflow
1. Define recovery targets and queue the request.
2. The agent packages the selection into an archive and uploads it.
3. Download or inspect the archive details from the controller list.
