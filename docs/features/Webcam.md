# Webcam

Turns on the remote webcam so operators can verify surroundings without leaving the console.

## Features
- Enumerates attached cameras and streams video frames in real time.
- Provides snapshots or continuous feed depending on the UI configuration.
- Optionally toggles resolution or frame rate when supported by the hardware.

## Basic Workflow
1. Open the Webcam tool and select the desired camera if multiples exist.
2. The agent streams encoded frames back to the controller for live viewing.
3. Stop the stream when the verification task completes.
